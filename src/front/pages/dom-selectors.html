DOM selectors

Inline selectors

You can attach CSS and JavaScript behaviors directly on tags. In this case, you don't need a selector for targeting the element you want to attach the behavior to.

Here we are using the style attribute to define the button style, and the onclick attribute to define the click event handler.

Notice the onclick attribute value: this refers to the button tag itself and event refers to the click event that occured.

Internal selectors

You can attach CSS and JavaScript behaviors using DOM selectors.

Here we define an id attribute that will allow us to select the tag.

In CSS, the id selector is #button2.

In JavaScript, the equivalent id selector is document.getElementById('button2'). Then we use the .addEventListener method to handle the click event. Notice that in the callback function this refers to the selected button.
